## API

#### 属性

属性 | 说明 | 类型 | 默认值
---|---|---|---
speed | 速度计算（如：每秒移动50px） | `Number` | 50
content | 内容 | `String, Function` | -
animated | 动画 | `Boolean` | true
autoplay | 未超出一屏时是否滚动 | `Boolean` | false

#### Slot

属性 | 说明
---|---
default | -


## 基础用法

```jsx
<vc-marquee content="222" />
```
